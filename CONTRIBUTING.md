# Contributing to MyThings

Thank you for your interest in MyThings!

As this is a proprietary project owned by **Ctrl Alt Win**, external contributions
are not currently accepted without prior written agreement.

## 🐛 Reporting Bugs

If you find a bug, please open an issue on GitHub with:

- A clear title and description
- Steps to reproduce the issue
- Expected vs actual behavior
- Your Android version and device model
- Screenshots if applicable

[Open an Issue](https://github.com/callixasta-ui/mythings/issues)

## 💡 Feature Requests

Have an idea? We'd love to hear it!

Open a GitHub issue with the label `feature request` and describe:
- What you'd like to see
- Why it would be useful
- Any examples or mockups
