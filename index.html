Get Started

Install in
4 easy steps.

No Play Store needed. Just download, allow, and install.

01

Download the APK

Tap the Download button above to get the latest APK file from GitHub Releases.

02

Allow Unknown Sources

Go to Settings → Security → Enable "Install from Unknown Sources" or tap "Allow" when prompted.

03

Open the APK

Find the downloaded file in your notifications or Downloads folder and tap to open it.

04

Tap Install

Follow the on-screen prompts and tap Install. MyThings will appear on your home screen.

⚠️ Note: MyThings requires Android 8.1 (API 27) or higher. If your device runs an older version, the app may not install correctly.
